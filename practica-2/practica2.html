# Etiqueta Definicion Ejemplo
1 !DOCTYPE tipo de documento indica al navegador la versión de html empleada
2 <!-- ... --> comentario (sólo visible en el código fuente) cuando necesitamos anotar algo
3 html engloba todo el documento contenedor de nuestra pagina
4 head delimita el encabezado del documento Contenedor del metadata
5 title título del documento. se muestra en la pestaña del navegador
6 base URI base para direcciones relativas Va en el header como link base de referencia.
7 link enlace a otros archivos Si queremos agregar un archivo que se encuentra en otra fuente
8 meta metainformación sobre el documento Cuando mandamos un link y carga imagenes y descripcion
9 style hoja de estilo incluida en el documento Para dar diseño a nuestra pagina
10 body delimita el cuerpo del documento Toda la estructura que podemos ver en la pagina
11 article La etiqueta especifica contenido independiente en un foro o blog
12 section Define una seccion del documento. Separar diseños
13 nav Define links de navegacion En un menu
14 aside Contenido aparte del principal Puede ser una pequeña observacion del contenido original
15 H1 Utilizado para crear un titulo en nuestro contenido principal el titulo del contenido de esta pagina
16 H2 titulo que le sigue a H1 Subtitulo
17 H3 titulo que le sigue a H2 Subtitulo
18 H4 titulo que le sigue a H3 Subtitulo
19 H5 titulo que le sigue a H4 Subtitulo
20 H6 titulo que le sigue a H5 y el final de los headers Subtitulo
21 header Introduccion de un articulo Dentro del tag article
22 footer Pie de pagina El contenido al final de las paginas que indica informacion generalmente de contacto
23 address Direccion Dirreccion de un local
24 p Párrafo Párrafos en un articulo
25 hr Separador de contenido Trazar una linea entre párrafo
26 div Encapsula elementos en su propio ambiente Separar algun diseño de otro
27 blockquote Cita larga Una cita que incluye varios párrafos
28 pre texto preformateado
29 main Contenido principal En una pagina de noticias este se usaria para resaltar una noticia
30 figure Similar al div pero se usa para separar imagenes Galeria de fotos
31 figcaption Dentro de figure como pie de foto Al final de una imagen para describirla
32 br salto de línea En un parrafo para crear un salto linea
33 wbr Posible salto de línea Si el texto es mas largo que la pantalla automaticamente da un salto
34 a hiperenlace un enlace para otro articulo o pagina
35 span Contenedor de texto genérico Agregar diseño a cierta parte del texto
36 strong Resaltar palabras o frases importantes Marcar texto a tomar en cuenta
37 em Enfatizar texto Para que el usuario haga enfasis en la lectura
38 sub subíndice Ejemplo la formula H2O
39 sup superíndice Ejemplo en 32
40 abbr abreviatura D.R
41 b Pone el texto en negrita sin hacerlo importante La practica 2 es muy larga
42 cite Citar una obra The Scream cuadro hecho por Edward Munch
43 data Ayuda a la maquina a leer datos Valores en elementos
44 dfn Definición de un elemento Para la definicion de un termino
45 kbd teclado F5 para actualizar la pagina
46 i tecnicismo Marcar texto en italic
47 mark Resaltado añadido posteriormente, no en el original Palabra o frase resaltado en un parrafo
48 q Definir una cita o frase La guerra es la información 90%
49 s Linea en medio Tickets agotados
50 small Texto mas pequeño de lo normal Puede ser algo no tan importante
51 time fecha y hora Horario Miercole 8:00 a 9:00
52 u Una palabra mal escrita marcada por una linea egemplo
53 code código mostrar un poco de codigo

55 var Variable matematica o programable Tel area del triangulo es: 1/2 * b * h
56 bdo Dirección de escritura Dirección de escritura(De derecha a izquierda)
57 bdi Ignorar dirección de escritura Un texto arababico por ejemplo no se veria afectado dentro del tag
58 applet Usado en HTML4 No soportado actualmente
59 map Usado para definir un mapa de imagenes Un motherboard con links en componentes
59 area Usado dentro de map para definir las imagenes Imagen de un escritorio con multiples herramientas
60 ins Texto insertado en el documento, marcado por una linea debajo este texto fue insertado
61 del Texto borrado en el documento, marcado por una linea en medio Este texto fue borrado
61 img Agregar una imagen Mostrar fotos en un blog
62 iframe marco incrustado en el documento Agregar un documento dentro de nuestro documento, en cuadro miniatura
63 embed Contenedor para un recurso externo imagenes, otros documentos, videos
64 object Define un contenedor para recursos externos, tal como iframe(recomendado) Imagenes, otros documentos, videos
65 param parámetro para <objeto> archivos de audios e imagenes dentro de object
66 video Usado para agregar un video a nuestro documento Agregar videos a nuestra pagina
67 source Usado para agregar un fuente de donde viene el video Agregar videos con ayuda de <video>
68 audio Tag usado para agregar sonidos Grabaciones en un podcast
69 track Especifica texto para los videos o audios de nuestra multimedia El texto del podcast se ira marcando a medida que avanza
70 ol Lista ordenada De 1 a N
71 ul lista no ordenada Inventarios
72 menu equivale a ul Menu en un restaurant
73 li elemento de lista (ordenada o no ordenada) Precios
74 dl lista de definición Un menu con platos y que incluye
75 dt término en lista de definición Ejemplo un platillo
76 dd definición en lista de definición Incredientes de un platillo
77 table Define una tabla Esta tabla que ves
78 caption leyenda de tabla Texto que describe la tabla
79 colgroup Grupo de columnas Columna de gastos

84 td Celda Elementos que hay en la fila de una tabla
85 th Celda de cabecera titulos de las columnas

87 form Define un formulario Ingresar información usuario o registrarse en una pagina
88 label Etiqueta de un control Etiqueta para un input
89 input control (hay varios tipos) Agregar rango,colores, texto, etc
90 button botón Boton para mandar un request
91 select caja de lista Crear una caja de lista
92 datalist Pre-define opciones para un <input> Listado de vehiculos
93 optgroup grupo de opciones en una caja de lista categorias dentro de un <select>
94 option opción de caja de lista En una lista de bebidas
95 textarea área de texto Una caja de comentarios
96 output Representa un calculo Numero de items que desea comprar
97 progress Representa una barra de completado 50%
